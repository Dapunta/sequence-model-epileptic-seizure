Deskripsi Data :
Dataset berisi sinyal electroencephalogram (EEG) untuk pengenalan kejang epilepsi. Versi yang umum dipakai di Kaggle/UCI merupakan hasil restrukturisasi dari Bonn EEG dataset, dengan total 11.500 sampel. Setiap baris merepresentasikan 1 segmen sinyal EEG dengan 178 titik data/atribut sebagai input sequence dan 1 kolom label sebagai output, sehingga totalnya 179 kolom. Dataset ini digunakan untuk multiclass classification dengan 5 kelas. Dataset asli terdiri dari 5 set, masing-masing berisi 100 file sinyal, dan tiap file berisi sekitar 4096/4097 titik sinyal. Pada versi UCI/Kaggle, tiap sinyal dibagi menjadi 23 segmen sehingga distribusinya menjadi relatif seimbang, sekitar 2300 sampel per kelas. Label target pada versi Kaggle/UCI dikodekan sebagai: 1 = seizure activity, 2 = EEG dari area tumor, 3 = EEG dari area otak sehat pada pasien epilepsi, 4 = healthy subject dengan mata tertutup, dan 5 = healthy subject dengan mata terbuka.
